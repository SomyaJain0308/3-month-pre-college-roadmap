## 🏁 Pre-Semester Final Checklist
*Everything that must be true on Day 1 at SST. Binary — yes or no.*

### Hard Skills
- [ ] Python — OOP, file I/O, error handling, common patterns (functional level)
- [ ] Data Structures — Arrays, Linked Lists, Stacks, Queues, Trees, Hash Maps implemented from scratch
- [ ] Algorithms — Binary Search, Sorting, BFS, DFS, Two Pointers, Sliding Window, intro to DP
- [ ] Linear Algebra — Vectors, matrices, eigenvalues, SVD intuition connected to NumPy
- [ ] ML — Linear Regression, Logistic Regression, K-Means, Decision Tree, PCA from scratch in NumPy
- [ ] SQL — Basic queries, JOINs, GROUP BY
- [ ] Web — HTML, CSS, basic JS (enough to build a static site)
- [ ] Git — Commits, branches, push/pull as daily habit
- [ ] Competitive Programming — 50+ Leetcode problems, 5+ Codeforces contests, rating 700–900

### Artifacts (Publicly Visible Things That Exist)
- [ ] Portfolio website live at `yourusername.github.io`
- [ ] 5+ GitHub repos with proper READMEs
- [ ] Technical blog with 2+ published posts
- [ ] LinkedIn profile complete and live
- [ ] Codeforces account with contest history visible
- [ ] 1 faculty email sent with a specific technical question

### Mindset Infrastructure
- [ ] Notion/Obsidian knowledge base with 12 weeks of notes
- [ ] Weekly tracking sheet habit established (updated every Sunday)
- [ ] Paper-reading habit initiated (at least 1 paper read and summarized)
